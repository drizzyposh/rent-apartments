# rent-apartments

A home market place where users can come and register and they can list their home or aparments for rent or for sale. They can slo add it as special offer. The design and layouts were built for mobile devices, but aswell we can use it on desktop. 
This project was built using React.js and Firebase where we get to:
- Register users
- Add/Save and delete users data to firestore database
- Sign in and Sign up, user log out
- Update users details in the database from firestore
- Update profile AUTH

Libraries used: React toastify to toast notifications alert on the pages.
